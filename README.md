# Java-Task-Manager
# Student task tracker

A java commandline application to help students manage tasks.
## Features
- Add task
- Delete task
- Mark task complete
- list all the tasks
- Search task by keyword
- Save tasks from the file
- Load tasks from file
- Priority Levels (High,Low)
- Due Dates

# Technologies used 
- Java
- Objected-Oriented Programming 
- Arraylist
- File I/O
